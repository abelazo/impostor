# Title
Add Participants to Game

# Description
As a game host
I want to add between 2 and 10 participants
So that we can play the game with our group

# Acceptance Criteria
- Display a form to add participants. No name required, just a label Player 1, Player 2, etc.
- Show current count of participants
- Limit maximum participants to 10
- Allow removing participants before starting
- Disable start button if less than 2 participants

# Implementation Notes
- Use a simple list or array to store participant names
- Use state management to handle adding/removing participants
- Validate participant count before enabling start button
- Basic styling for participant list and buttons
- No backend integration needed for this user story
